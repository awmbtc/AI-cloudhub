# 薅羊毛版：控制面服务器 / DB / Redis

> 原则：**能同机就同机、能白嫖就白嫖、能年付就年付。**  
> 产品是 **AI-cloudhub**（Go 自研；非 MinIO 二开；不托管用户对象账单）。  
> 架构定稿见 [ARCHITECTURE.md](./ARCHITECTURE.md)。

---

## 0. 先泼冷水

| 你能薅到的 | 你薅不到的 |
|------------|------------|
| 控制面（API+PG+Redis）可以 **年付一两百～小几百** 跑起来 | 用户自己在 R2 存几百 TB —— 账单在用户侧，不是我们服务器 |
| 我们 API 几乎不传文件 body | 若错误做成中转网盘，带宽照样爆 |

**Drive Map 适合薅羊毛：API 极瘦；文件直写用户对象存储。**

## 1. 白嫖 / 极低价清单（组件级）

### 1.1 服务器（计算）

| 来源 | 常见羊毛规格 | 量级价格 | 备注 |
|------|--------------|----------|------|
| **腾讯云 / 阿里云 轻量** | 2核2G / 2核4G，年付 | **约 99～200 元/年**（活动） | 新客/学生/秒杀；流量包月含在套餐里 |
| 轻量进阶 | 2核4G / 4核4G | **约 200～500 元/年** | 比按量 ECS 便宜一个数量级 |
| 腾讯轻量对象存储赠送 | — | **约 50GB** 随锐驰型等套餐 | 有就白嫖，无则当没有 |
| **Oracle Cloud 永久免费** | ARM 4OCPU / 24G 等（额度内） | **$0** | 难注册、易封号，能上车当送的 |
| **Hetzner / 搬瓦工等** | 2C4G～4C8G | **约 €4～10/月** | 海外；挂国内用户延迟看业务 |
| 多开小号 / 多云各买一台年付 | 2× 轻量 | **约 200～400 元/年** | 一台 API+DB，一台纯备用/MinIO |

**默认推荐（国内业务）：**  
**1 台轻量 2核4G 年付** → 全栈 docker（api + postgres + redis + 可选 minio）。

### 1.2 数据库（别买云 RDS）

| 方案 | 价格 | 说明 |
|------|------|------|
| **同机 PostgreSQL Docker** | **¥0**（含在服务器里） | **羊毛首选**；记得每天备份到对象存储 |
| SQLite（更早期 MVP） | **¥0** | 单机、低并发够用；上用户再迁 PG |
| Neon / Supabase 免费层 | **$0** | 有额度与休眠限制，适合开发 |
| 云 RDS 包年 | 动辄几百～上千/月 | **不薅这个** |

### 1.3 Redis

| 方案 | 价格 | 说明 |
|------|------|------|
| **同机 Redis Docker** | **¥0** | **羊毛首选**；限流+session 完全够 |
| Upstash Redis 免费层 | **$0** | 有命令数限制，API 在海外时好用 |
| 云数据库 Redis | 按量/包月 | **早期不买** |

### 1.4 对象存储（真正要动脑的）

| 方案 | 免费/便宜点 | 超出后 | 适合 |
|------|-------------|--------|------|
| **Cloudflare R2** | **10GB 免费**；**出口流量 $0** | 存储约 **$0.015/GB·月**（约 **0.11 元/GB·月**） | **强烈推荐**；S3 兼容，免下行杀猪 |
| Backblaze B2 | 免费额度看活动 | 存储便宜，出口配合 CF 可优化 | 备选 |
| 腾讯/阿里 OSS 资源包 | 新客 **40G～100G 年包几十～百元**；500G 活动约 **百元级/年** | 按量存储+**下行贵** | 仅国内合规/延迟硬需求 |
| 轻量自带盘当 MinIO | 盘在轻量价里 | 盘满就扩容/换机 | 数据 < 套餐盘容量时最省 |
| 轻量赠送对象 50GB | 有则白嫖 | — | 有就用 |

**羊毛最优对象策略：**

```text
开发 / 早期：   轻量本机 MinIO（用系统或数据盘） 或 R2 免费 10G
用户上来：     Cloudflare R2（免出口）当主存储
必须国内：     OSS 只买存储包，能内网就内网；公开下载走 CDN 流量包
```

## 2. 推荐拓扑（穷鬼三档）

### 档 ① 极致穷：一年两位数～小一百（验证）

```text
┌─────────────────────────────────────┐
│  轻量 2C2G 或 2C4G（年付 99～200）   │
│  docker compose:                    │
│    - api                            │
│    - postgres（或先 sqlite）         │
│    - redis                          │
│    - minio（数据放 /data 盘）        │
└─────────────────────────────────────┘
```

| 项 | 配置 |
|----|------|
| CPU/内存 | **2核 2G** 能跑通；舒服点 **2核 4G** |
| 盘 | 套餐自带 **40～80G**；能加挂便宜盘就加 |
| 带宽 | 轻量套餐自带 **峰值带宽 + 月流量包**（通常够演示） |
| DB/Redis | **同机，不另买** |
| 对象 | 同机 MinIO 或 R2 免费档 |

**月均成本：约 8～20 元/月（按年付摊）**  
**容量：别吹 10 万用户存 TB 级**——这档是产品验证。

---

### 档 ② 正经穷：一年两三百～小一千（小规模真用）

```text
┌─ 轻量 A：2C4G 年付 ──────────────┐
│  api × 1～2（同机多进程也行）      │
│  postgres + redis                  │
└────────────────────────────────────┘
              │ S3 API
┌─────────────▼──────────────────────┐
│  Cloudflare R2（主）                │
│  或 轻量 B：2C2G 只跑 MinIO + 大盘  │
└────────────────────────────────────┘
```

| 角色 | 羊毛规格 | 年费量级 |
|------|----------|----------|
| 应用机 | 轻量 **2核4G** | **约 150～300 元/年** |
| 对象 | **R2** 按量；前 10G 免费 | 存 100GB ≈ **$1.5/月 ≈ 11 元/月** 量级 |
| 或 MinIO 机 | 再买一台轻量 2C2G + 扩容盘 | **约 100～200 元/年 + 盘** |
| DB/Redis | 仍同机 A | **0** |

**月均：约 30～80 元（控制面+小对象）**；对象随容量线性加（R2 无出口费很关键）。

---

### 档 ③ 穷但能撑「很多账号」（元数据 10 万级，存储另算）

> 10 万 **注册** + 路径直传 + 对象在 R2/OSS：控制面仍可以很穷。

```text
轻量/ECS 年付 2C4G × 2（API 主备或 API / DB 分离）
  机1：api + redis
  机2：postgres（盘大一点）
对象：Cloudflare R2（免出口）
备份：pg_dump 定时丢 R2
CDN：Cloudflare 免费层（静态/部分下载）
```

| 角色 | 规格 | 数量 | 年费量级 |
|------|------|------|----------|
| API | 轻量 **2C4G** | 1～2 | **150～400 元/年** 合计 |
| PG | 轻量 **2C4G** 或与 API 同机 | 1 | 含上或 **+100～200 元/年** |
| Redis | **同 API 机** | — | **0** |
| 对象 R2 | 按实际 GB | — | 见下表 |
| 域名+CF | 域名 ~50 元/年 | — | SSL 免费 |

**控制面年费目标：约 300～800 元/年（约 25～70 元/月）**  
**不要买：** 云 RDS、云 Redis、高防、按量 ECS 乱开带宽。

## 3. 对象存储「穷算」表（R2 口径）

R2：存储约 **$0.015/GB·月**，**出口 $0**（相对国内 OSS 最大羊毛）。

| 总存储 | 约 美元/月 | 约 人民币/月 |
|--------|------------|--------------|
| 10 GB（免费内） | 0 | **0** |
| 100 GB | ≈ 1.5 | **≈ 11** |
| 1 TB | ≈ 15 | **≈ 110** |
| 10 TB | ≈ 150 | **≈ 1,100** |
| 100 TB | ≈ 1,500 | **≈ 11,000** |

对比：国内 OSS 100TB 存储约 **1 万+/月**，再加下行更疼。  
**能接受数据走国际链路 / R2 的，这是降本最大刀。**

国内硬要求时：OSS **只买存储包**，公开下载走 **CDN 流量包**，Agent 尽量同云内网。

## 4. 穷鬼配置 vs 我之前「正规配置」对照

| 项 | 正规（贵） | **羊毛（本页）** |
|----|------------|------------------|
| API | 2C4G × 3～4 按月 | **轻量 2C4G × 1～2 年付** |
| PG | 4C16G RDS | **同机 Postgres Docker** |
| Redis | 独立 2C4G / 云 Redis | **同机 Redis Docker** |
| 对象 | 国内 OSS 按量 | **R2 或 本机 MinIO** |
| LB | 云 SLB | **Caddy/Nginx 本机 + CF** |
| 10 万用户控制面 | 1.2k～3.5k/月 | **约 25～70 元/月** |
| 100TB 存储 | OSS ~1.2 万/月 | **R2 ~1.1 万/月**（仍贵，但是出口不杀） |

存储上 TB 后，**谁都不可能免费**——只能换更便宜的存储单价（R2/B2）和砍配额。

## 5. 直接抄作业：购物车

### 现在就干（最低）

```text
□ 腾讯云/阿里云 轻量 2核4G 年付（有 2C2G 秒杀也可先上）
□ 域名随便一个（可后期再绑）
□ Cloudflare 账号（R2 + 代理可选）
□ 本机：docker compose 起 api + postgres + redis + minio
□ 备份脚本：pg_dump | 上传 R2/MinIO 另一桶
```

**预算：约 100～200 元/年 + 域名**

### 有点用户了

```text
□ 应用仍 1～2 台轻量年付
□ 对象迁到 Cloudflare R2（S3 兼容，改 endpoint）
□ 配额做严：默认 100MB～1GB/人
□ 监控：Uptime 免费 + 磁盘告警脚本
```

### 号称冲 10 万注册（穷版）

```text
□ 控制面：2 台 2C4G 轻量年付（API / PG 分开更稳）
□ Redis：继续同 API
□ 对象：R2，人均配额卡死
□ 不做：云数据库、云 Redis、API 中转文件、一用户一桶乱开
```

**控制面预算：约 300～800 元/年**  
**对象：按实际上报，用上面 R2 表自己乘。**

## 6. 同机资源怎么切（2C4G 轻量）

| 进程 | 内存建议 | CPU |
|------|----------|-----|
| postgres | 512MB～1GB | 共享 |
| redis | 64～128MB | 共享 |
| api | 128～256MB × 1～2 | 共享 |
| minio（若同机） | 256～512MB | 共享 |
| 系统+docker | 余量 | — |

2C2G 更紧：优先 **api + sqlite + 无 redis（或内存限流）+ R2 不装 MinIO**。

## 7. 一句话价格（你要的「预计多少钱」羊毛版）

| 阶段 | **预计** |
|------|----------|
| 验证上线 | **约 10～20 元/月**（年付轻量摊销）或 **首年 100～200 元** |
| 小规模真用 + 对象几百 GB | **控制面 ~30 元/月 + R2 约 几十元/月** |
| 10 万注册、配额很狠、R2 存 1TB 级 | **控制面 ~50 元/月 + 对象 ~100 元/月 → 合计约 150 元/月量级** |
| 10 万用户人均 5GB（500TB） | **对象 alone ~千元～万元/月**——这和羊毛无关，是物理容量 |

## 8. 禁忌（穷也会死）

1. 买了轻量又去开 **按量 5M 带宽的 ECS 中转下载**  
2. 早期上 **云 RDS + 云 Redis**（月费超过服务器年费）  
3. 无备份（同机盘挂了数据全没）  
4. 不设配额，被当免费网盘盘  
5. 以为「10 万用户」= 必须 4C16G×N——**元数据不需要，对象才需要钱**
